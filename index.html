How to Enable APIs
  • Go to chrome://flags/#language-detection-api and chrome://flags/#translation-api
  • Select Enabled
  • Click Relaunch or restart Chrome
Your browser doesn't support the Translator or Language Detector APIs. Please use Chrome with the Early Preview Program enabled.

About This Project

Learn more about the technology and vision behind this playground.

This project showcases the capabilities of Chrome's experimental Language Detector and Translator APIs. Created by Eno-obong Umoh, it aims to provide an accessible and interactive way to explore AI-driven language processing. Whether you're a developer, educator, or language enthusiast, this tool is designed to inspire and educate.

Powered by Google's Chrome AI APIs, this playground is part of the Early Preview Program, offering a glimpse into the future of web-based AI applications.
